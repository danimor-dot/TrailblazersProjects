## Submission Guide

- Please submit your complete App.js to this folder.
- Rename the file as the following before upload: **`{first name}_{last_name}_App.js`**.
- For example: **`Bakari_Kerr_App.js`**
- The code in the file must reflect information about yourself as a student.
- Additionally, even though you are allowed to copy and paste from the exemplar, you must modify more than just the text.
- AI tools such as Github Copilot, Cursor, and ChatGPT are permitted and encouraged.
- The usage of additional libraries such as REST APIs, Bootstrap, Material-UI, and Tailwind CSS are also recommended.
